## Features

- Rsbuild
- React
- React Router v6
- Tailwindcss
- ESLint and Prettier

## Setup

Use pnpm or any other package manager:

```bash
pnpm dlx degit emrocode/reactx#main ./my-app
```

```bash
cd ./my-app && pnpm install && pnpm run dev --open
```
